# Агент обработки фото для RitualB2B

Принимает фото товара через Telegram, превращает их в премиальную карточку через
ChatGPT (image generation), сохраняет результат в `output/` и отдаёт обратно в Telegram.

## Что делает агент

1. Ты пишешь боту в Telegram → присылаешь фото товара
2. Бот сохраняет фото в `input/`, ставит в очередь
3. Скрипт автоматически:
   - открывает новый чат в ChatGPT (через специальный браузер)
   - вставляет 2 эталона стиля + твоё фото + промпт по ТЗ
   - ждёт результат (~1–2 мин)
   - скачивает PNG, кладёт в `output/`
4. Бот присылает готовую карточку обратно в Telegram
5. Исходный файл уезжает в `processed/` (архив)

## Структура

```
agent_convert_foto_rituailb2b2/
├── reference/          эталоны стиля + tz.txt
├── input/              входящие фото (очередь)
├── output/             готовые карточки (по дате-времени)
├── processed/          архив обработанных исходников
├── logs/               логи агента и бота
├── chrome_profile/     отдельный Chrome-профиль (логин в ChatGPT)
├── agent.py            обработка одного файла (Playwright + Chrome)
├── bot.py              Telegram-бот + worker очереди
├── config.py           настройки (читает .env)
├── clipboard_utils.py  копирование картинок/текста в clipboard Windows
├── start_chrome.bat    запуск Chrome для агента
├── start_bot.bat       запуск бота
└── .env                твои токены (не в git)
```

## Первая настройка (один раз)

### 1. Установить зависимости (если ещё нет)

```cmd
pip install -r requirements.txt
```

### 2. Получить Telegram bot token

1. В Telegram открой [@BotFather](https://t.me/BotFather)
2. `/newbot` → имя `Ritual Photo Bot` → username (например `ritual_photo_processor_bot`)
3. Скопируй токен вида `1234567890:ABCdef...`

### 3. Узнать свой Telegram user ID

1. В Telegram открой [@userinfobot](https://t.me/userinfobot)
2. Он покажет твой `Id` (число вроде `123456789`)

### 4. Создать `.env`

Скопируй `.env.example` → `.env` и заполни:

```
TELEGRAM_BOT_TOKEN=1234567890:ABCdef...
TELEGRAM_ALLOWED_USER_ID=123456789
```

(Только этот ID сможет писать боту.)

### 5. Залогиниться в ChatGPT (один раз)

1. Двойной клик: **`start_chrome.bat`**
2. Откроется отдельное окно Chrome с chatgpt.com
3. Войди в свой ChatGPT Plus аккаунт
4. Профиль сохранится в `chrome_profile/` — больше логиниться не нужно

## Каждодневное использование

1. Двойной клик: **`start_chrome.bat`** (если ещё не запущен)
2. Двойной клик: **`start_bot.bat`** (или `python bot.py` в терминале)
3. Открой своего бота в Telegram
4. Команда `/start`
5. Шли фото — получай карточки

Результаты лежат в `output/` с именами вроде `ritual_2026-04-26_15-30-00_001.png`.

## Команды бота

- `/start`, `/help` — справка
- `/status` — сколько в очереди / сколько готово

## Замены эталонов или ТЗ

- Эталоны — `reference/etalon_1.png` и `reference/etalon_2.png`
- ТЗ (промпт) — текст в `config.py`, переменная `PROMPT_TEMPLATE`

## Что важно знать

- ⚠️ Автоматизация веб-интерфейса ChatGPT нарушает ToS OpenAI. Аккаунт под риском
  при массовой/быстрой обработке. Не отправляй сотни фото подряд.
- ⚠️ Кнопку отправки в ChatGPT нажимаем после того, как все 3 фото догрузились.
  Это занимает ~30 сек на медленном интернете.
- ⚠️ Chrome для агента (тот, что запускает `start_chrome.bat`) лучше не трогать
  во время обработки — может сбросить focus.
- ⚠️ Кириллический текст на лентах товара ChatGPT иногда пишет с ошибками — это
  лимит модели, не баг агента.
- ⚠️ Финальное разрешение результата ~1254×1254. Если нужно строго 4K —
  доп. шаг апскейла (Real-ESRGAN или похожее) можно добавить позже.

## Проблемы

- **`CDP not ready`** при запуске агента → `start_chrome.bat` не запущен.
- **`send-button disabled`** → ChatGPT ещё грузит вложения. Агент ждёт до 60 сек.
- **Бот не отвечает** → проверь `.env`, что токен правильный, и что сам ты пишешь
  с того аккаунта, чей ID указан в `TELEGRAM_ALLOWED_USER_ID`.
- **Результат не похож на эталон** → отредактируй промпт в `config.py:PROMPT_TEMPLATE`.
